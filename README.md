# LeetStats

## Description

LeetStats is a web application that helps users track and visualize their LeetCode performance by simply entering their LeetCode username. The application fetches real-time data from the LeetCode GraphQL API and presents key statistics such as the number of Easy, Medium, and Hard problems solved, total submissions, and overall coding progress.

The project features an interactive and responsive user interface with circular progress indicators that dynamically update based on the user's problem-solving data, providing a quick and intuitive overview of their LeetCode journey.

## Features

* Search LeetCode profiles by username
* Fetch real-time LeetCode statistics
* Display Easy, Medium, and Hard problem counts
* Visualize progress using circular progress indicators
* Show overall submission statistics
* Responsive and interactive UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GraphQL
* LeetCode API
* CORS Anywhere Proxy

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the project using Live Server or any local web server.
4. Enter a valid LeetCode username and click **Search**.
